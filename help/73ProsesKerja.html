  1. 7 Modul TA Regional Economy-Descriptive Analysis

7.3 Proses Kerja


Proses Kerja

  1. Membuka jendela sub-Modul Descriptive Analysis, dengan memilih ikon Modul TA→Regional Economy→Descriptive analysis (Gambar 7.1), hingga terbuka jendela analisis seperti pada Gambar 7.2.

Gambar 7.76 Menu TA


Gambar 7.77 Jendela analisis TA Regional Economy sub-modul Descriptive analysis


  1. Mengisi kolom Intermediate consumption matrix

Kolom ini diisi dengan data tabel keterkaitan antar sektor ekonomi yang ada di suatu daerah. Pilih tombol → Intermediate_Demand (Gambar 7.3)


Gambar 7.78 Pemilihan data untuk mengisi kolom Intermediate consumption matrix

  1. Mengisi kolom Value addes matrix

Kolom ini diisi dengan menggunakan data tabel matrix nilai tambah, yang menunjukkan nilai tambah dari masing-masing sektor ekonomi. Pilih tombol → Value_added (Gambar 7.4)


Gambar 7.79 Pemilihan data untuk mengisi kolom Value addes matrix


  1. Mengisi kolom Final consumtion matrix

Kolom ini diisi dengan menggunakan data tabel matrik permintaan akhir, yang menunjukkan jumlah permintaan akhir terhadap output yang diproduksi dari masing-masing sektor. Pilih tombol → Final_Demand (Gambar 7.5)


Gambar 7.80 Pemilihan data yang digunakan untuk mengisi kolom Final consumtion matrix



  1. Mengisi kolom Value added component

Kolom ini diisi dengan menggunakan data tabel komponen nilai tambah, yaitu komponen yang berkontribusi terhadap nilai tambah keseluruhan sektor ekonomi. Pilih tombol → Value_Added_Component (Gambar 7.6)


Gambar 7.81 Pemilihan data yang digunakan untuk mengisi kolom Value added component



  1. Mengisi kolom Final consumption component

Kolom ini diisi dengan menggunakan data tabel yang berisi komponen permintaan akhir. Pilih tombol → Final_Demand_Component (Gambar 7.7)


Gambar 7.82 Pemilihan data yang digunakan untuk mengisi kolom Final consumtion matrix



  1. Mengisi kolom List of economic sector

Kolom ini diisi dengan menggunakan data tabel yang berisi daftar sektor ekonomi pada daerah yang akan dianalisis. Pilih tombol → Economic_Sector (Gambar 7.8)


Gambar 7.83 Pemilihan data yang digunakan untuk mengisi kolom List of economic sector


  1. Mengisi kolom Labour requirement  

Kolom ini diisi dengan menggunakan data tabel kebutuhan tenaga kerja dari masing-masing sektor ekonomi. Pilih tombol → Labour (Gambar 7.9)


Gambar 7.84 Pemilihan data yang digunakan untuk mengisi kolom Labour requirement


  1. Mengisi kolom Financial Unit

Kolom ini diisi dengan satuan moneter yang diinginkan.


  1. Mengisi kolom Areal name

Isi kolom Areal name dengan menuliskan nama area yang dianalisis.


  1. Mengisi kolom Year

Isi kolom Year dengan menuliskan angka tahun yang ingin dianalisis.


Gambar 7.85 Pengisian kolom Year


  1. Menjalankan proses analisis TA-Regional Economy sub-modul Descriptive analysis

Setelah seluruh kolom terisi dengan benar, mulai proses analisis dengan memilih tombol Process. Tunggu hingga proses analisis selesai, yang ditandai dengan munculnya jendela seperti Gambar 7.11

Gambar 7.86 Jendela yang menunjukkan bahawa proses analisis telah selesai
